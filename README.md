# Fractals

This repository contains example fractal generators:

- mandelbrot.py — Generates a Mandelbrot set image using numpy + Pillow.
- sierpinski.py — Generates a Sierpinski triangle (chaos game) using Pillow.
- fractal.html — Interactive Mandelbrot renderer (open in a browser).

Requirements:
- Python 3.8+
- For mandelbrot.py: numpy & pillow
  - Install: `pip install numpy pillow`
- For sierpinski.py: pillow
  - Install: `pip install pillow`

Usage:
- Run `python mandelbrot.py` to produce `mandelbrot.png`.
- Run `python sierpinski.py` to produce `sierpinski.png`.
- Open `fractal.html` in any modern browser to explore interactively.
